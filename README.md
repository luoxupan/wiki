### Mac启动自带服务器实现本地访问
- `sudo apachectl start`
- `sudo apachectl restart`
- `sudo apachectl stop`
- 修改服务器默认配置
  - `sudo vim /etc/apache2/httpd.conf`
  - 将`<Directory "/Library/WebServer/Documents">`修改成`<Directory "/Users/didi">`
- 浏览器输入`http://localhost/wiki/pages/page06/index.html`
